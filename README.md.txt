1.# Guia Git Colaborativo
2.Este repositório contém um resumo colaborativo dos principais conceitos de Git e GitHub.

*/#region Conceitos Basicos Git ##*/

    1 - Reposiório
        Repositorio é onde seu condigo é armazenado (offline e online)
    2 - Commit
        Commits são feitos para descrever o que foi alterado no código
    3 - Branch
        Branchs são usados para separar features da aplicação, evitando conflitos entrecolaboradores
            trabalhando nos mesmos códigos
    4 - Merge
        Merge é usado para fundir Branches entre si ou com o Main (master)

*/#endegion#*/
## Comandos Essenciais
1.- git clone
2.- git status
3.- git add
4.- git commit
5.- git push
6.- git pull
